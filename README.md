# Modèle

## Documenter son projet

Le dossier technique doit contenir les éléments précisés ci-dessous, en respectant cette même nomenclature :

- un fichier README.md,
- un fichier ~licence.txt~ LICENSE,
- un répertoire sources,
- un répertoire docs,
- un fichier requirements.txt,
- un répertoire data (optionnel),
- un répertoire test (optionnel),
- un répertoire exemples (optionnel),
- un fichier presentation.pdf.


Le dossier technique est à déposer sur la forge des communs numériques éducatifs.
